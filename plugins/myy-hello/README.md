# myy-hello

示例插件 - Hello World skill。

## 包含的 Skills

| Skill | 调用方式 | 说明 |
|-------|---------|------|
| hello | `/myy-hello:hello` | 简单的 Hello World 问候 |

## 安装

此插件作为 `myy-cc-plugins` 市场的一部分发布，安装市场后即可使用：

```
/plugin marketplace add myy-claw/plugin-myy-dev
```

也可单独加载进行本地测试：

```
claude --plugin-dir ./plugins/myy-hello
```
