# Translate Skill

中英文互译工具，特别适合技术文档和代码注释的翻译。

## 使用方式

用户通过 `/myy-demo-b:translate` 调用此 skill。

## 行为

### 指令

- 读取 `$ARGUMENTS` 中的内容（文本、文件路径或上下文中的代码）
- 自动检测语言：中文内容翻译为英文，英文内容翻译为中文
- 如果用户在 `$ARGUMENTS` 中显式指定了目标语言（如 "翻译成日文"），按指定语言翻译
- 翻译规则：
  - 技术术语保留原文并在括号中注明翻译，如 "middleware（中间件）"
  - 代码中的变量名、函数名不翻译
  - 保持原文的格式和结构（Markdown、列表等）
  - 代码注释翻译时保持注释格式不变
