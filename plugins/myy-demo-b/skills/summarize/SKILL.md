# Summarize Skill

将长文本、文件或代码总结为简洁的要点。

## 使用方式

用户通过 `/myy-demo-b:summarize` 调用此 skill。

## 行为

### 指令

- 读取 `$ARGUMENTS` 中指定的内容（可以是文件路径、粘贴的文本、或上下文中的代码）
- 生成结构化的摘要，包含：
  1. **一句话总结** - 用一句话概括核心内容
  2. **关键要点** - 3-5 个要点，使用列表格式
  3. **细节补充**（可选） - 如果内容较复杂，补充重要细节
- 如果是代码文件，侧重总结：功能用途、核心逻辑、依赖关系
- 如果是文档/文章，侧重总结：主题、论点、结论
- 输出语言与原文保持一致，如无法判断则使用中文
