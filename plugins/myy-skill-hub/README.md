# myy-skill-hub

技能市场维护工具集，用于 skills-map 门户的案例制作与管理。

## 包含的 Skills

| Skill | 调用方式 | 说明 |
|-------|---------|------|
| skills-map-skill-builder | `/myy-skill-hub:skills-map-skill-builder` | 为 skills-map 门户制作实践案例 |

## 安装

此插件作为 `myy-cc-plugins` 市场的一部分发布：

```
/plugin marketplace add myy-claw/cc-plugins
```

也可单独加载进行本地测试：

```
claude --plugin-dir ./plugins/myy-skill-hub
```
